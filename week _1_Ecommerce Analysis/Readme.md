# Week 1: E-commerce Revenue Analysis

## Objective
Analyze transaction data to understand what drives revenue.

## Dataset
UCI Online Retail dataset

## Key Steps
- Data cleaning (missing values, duplicates, inconsistencies)
- SQL analysis
- Python validation
- Dashboard creation

## Key Insights
- 84% of revenue comes from the UK
- Revenue concentrated among few customers
- Postage entries distort product analysis
- High sales ≠ high return rate (needs rate analysis)

## Tools Used
- Python (pandas)
- SQL (SQLite)
- Excel

## Files
- notebooks/ → analysis
- data/ → cleaned data
- dashboard/ → Excel dashboard
- sql/ → queries
- summary/ → conclusions
