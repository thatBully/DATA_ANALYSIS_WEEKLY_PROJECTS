# Data Analytics Portfolio

This repository contains my weekly data analysis projects, where I apply real-world datasets to build skills in data cleaning, SQL, Python analysis, and dashboarding.

## Objective

To develop strong analytical thinking and practical data skills by completing end-to-end projects involving:

- Data cleaning and preprocessing
- SQL-based analysis
- Python (pandas) for data exploration
- Data visualization
- Dashboard creation (Excel / Power BI)
- Business insight generation

## Projects

### Week 1: E-commerce Revenue Analysis
- Analyzed transactional retail data to understand revenue drivers
- Identified geographic concentration (84% revenue from the UK)
- Detected customer revenue concentration
- Highlighted issues with product-level analysis (e.g., postage entries)
- Built a dashboard to visualize key metrics

*(More projects will be added weekly)*
